The Idea

A real fighter. An unreal input.

Meowdown is a 1v1 voice fighter built for mobile and Steam from day one. The mouth is the controller. The cat is the fighter. Eight moves, 30-second matches, ranked play — across platforms, in the same matchmaking pool. Phone fights PC. Belgrade fights Berlin. All in the same fight. This page is the open notebook on what it is, how it works, and where it's headed.

/ 01 /
VOICE

No buttons. Meow, hiss, purr, and yowl are the only inputs. The mouth is the controller.

/ 02 /
VIRAL

Every match produces an embarrassing clip. The viral loop is built into the input itself.

/ 03 /
VERSUS

30-second matches, best of 3, ranked play across platforms. Real combat: parries, clashes, mind games.

/ 04 /
EXPAND

Same engine ships roosters, geese, and frogs next year. Content roadmap built into the architecture.

The Combat

Eight moves. One mouth.

A real fighting game underneath the comedy. Same-move clashes prevent spam. Parries reward skill. Yowls reward courage. Loudness alone never wins.

Claw Swipe
SHORT MEOW

Fast jab. Low damage, low cost. Spam is punished.

Pounce
LONG MEOW

Heavy committed leap. Breaks guard. Punishable on whiff.

Hiss Block
LONG HISS

Sustained guard. Stops jabs. Broken by Pounce.

Sharp Parry
SHARP HISS

Tight 250ms window. Counter-hit + stun. The skill ceiling.

Silent Dodge
SILENCE

No sound, no cost. Avoids everything. Reading required.

Purr Focus
LOW HUM

Recover stamina, charge ultimate. Exposes you.

Scratch Flurry
RAPID MEOWS

Three quick hits. Breaks blocks. Caught by parry.

Ultimate Yowl
FULL METER + LOUD

Unblockable. 3 damage. Only Silent Dodge counters it.

The Input

What your mic sees. What the cat does.

No language model. No keyword spotting. The game listens for shapes of sound — short / long / sharp / silent / sustained — and maps them to moves. Here's the acoustic grammar.

"meow!"
→
Claw Swipe
"meeeeow"
→
Pounce
"hsssss"
→
Hiss Block
"tss!"
→
Sharp Parry
"mrrrrr"
→
Purr Focus
— silence —
→
Silent Dodge
"MRAAAAOWW!"
→
Ultimate Yowl

The game classifies intent across six acoustic dimensions — short/long/voiced/unvoiced/loud/silent. You don't need to be a cat. You need to be a different cat each time.

In the air

Things I'm still figuring out.

The page wouldn't be honest without these. If any of them sparks something, that's exactly the kind of email I'm hoping for.

/ 01 / Tone

Cult of the Lamb dark-comedy, or Cuphead chaotic-cute? Both work. Both pull different audiences.

/ 02 / Music

Cat Noir Lounge jazz is the brief. AI-generated for MVP? Commissioned composer post-launch? Open to applicants.

/ 03 / Mascot

One hero cat across all marketing, or a rotating breed-of-the-month? The first builds franchise. The second builds variety.

/ 04 / Announcer

Snarky British documentarian, decided. But — recorded human voice actor, ElevenLabs AI, or a cat-voiced narrator? I genuinely don't know.

/ 05 / Name

"Meowdown" feels right but the trademark check is pending. Backup names on the shelf: Catstrophe, Yowl, Hiss & Co., Furrocious, Alley Royale.

/ 06 / Soft-launch market

Belgrade first feels right for the city-leaderboard hook. But does it work as well in Berlin? Mexico City? Manila? TBD.
