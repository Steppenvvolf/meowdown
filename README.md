# Meowdown — Idea Showcase

The 1v1 voice fighter where your meow is the controller. Mobile & Steam, cross-platform, Unity 6.

**Live page:** https://steppenvvolf.github.io/meowdown/

This repository hosts a single-page idea showcase for **Meowdown**, currently in pre-production at Triardor DOO (Belgrade). It is open to collaboration — animators, composers, Unity engineers, playtesters, streamers, and investors are all welcome to reach out.
